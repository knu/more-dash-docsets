describe-cluster¶

Description¶

Provides cluster-level details including status, hardware and software configuration, VPC settings, bootstrap actions, instance groups and so on. For information about the cluster steps, see <code>list-steps</code>.

Synopsis¶

  describe-cluster
--cluster-id <value>

Options¶

--cluster-id (string)

A unique string that identifies the cluster. This identifier is returned by create-cluster and can also be obtained from list-clusters .

Examples¶

  • Command:

    aws emr describe-cluster --cluster-id j-XXXXXXXX
