[ aws . gamelift ]

upload-build¶

Description¶

Upload a new build to AWS GameLift.

Synopsis¶

  upload-build
--name <value>
--build-version <value>
--build-root <value>

Options¶

--name (string) The name of the build

--build-version (string) The version of the build

--build-root (string) The root directory of build to upload
